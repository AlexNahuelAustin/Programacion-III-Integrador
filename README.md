### Características principales

- **Catálogo dinámico:** Visualizar productos por categorías, buscador integrado
- **Carrito de compras:** Agregar/quitar items, calcular totales con envío fijo
- **Autenticación:** Login/registro de usuarios con roles (ADMIN/USUARIO)
- **Historial de pedidos:** Cada cliente puede ver sus pedidos confirmados
- **Panel administrativo:** Visualizar estadísticas, gestionar categorías/productos/pedidos
- **Gestión CRUD:** Alta, baja, modificación de categorías, productos y pedidos
- **Validación de Stock:** Descuento automático y validación de disponibilidad
- **Reportes:** Consultas JPQL de productos, pedidos y totales facturados
- **Persistencia de Datos:** localStorage (frontend) + H2 Database (backend)
-----

### Tecnologías usadas

**Frontend:** TypeScript, Vite, HTML5, CSS3  
**Backend:** Java 21, JPA, Hibernate, H2, Maven
-----

## Sobre el Proyecto

Este TPI demuestra el dominio de una arquitectura full-stack completa. Durante el desarrollo se 
aplicaron patrones de diseño como DTO, Repository Pattern y transacciones atómicas. El proyecto 
aborda desafíos reales como validación de datos, protección de rutas, manejo de errores y persistencia 
segura. FoodStore constituye una base sólida que puede extenderse con APIs reales, sistemas de pago 
e integración con servicios externos.
